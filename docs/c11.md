Already Implemented

- _Generic (type-generic selection)
- _Alignas / _Alignof (alignment)
- _Atomic (atomic types)
- _Thread_local (thread-local storage)
- _Noreturn (function specifier)
- _Bool
- Anonymous structs/unions
- u8 string literals

Missing / Could Add

1. _Static_assert - Compile-time assertions
_Static_assert(sizeof(int) == 4, "int must be 4 bytes");
1. Relatively easy to add - just evaluate a constant expression at parse time and emit an error if false.
2. _Complex / _Imaginary - Complex number types
double _Complex z = 1.0 + 2.0*I;
2. More involved - requires new types and arithmetic operations.
3. Unicode string literals u"" and U"" - char16_t/char32_t strings
char16_t *s16 = u"hello";
char32_t *s32 = U"hello";
4. char16_t and char32_t types - Fixed-width character types
5. __STDC_NO_*__ macros - Feature test macros to indicate missing optional features
6. VLA improvements - C11 made VLAs optional but added __STDC_NO_VLA__
