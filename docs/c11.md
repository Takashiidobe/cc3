# C11 Feature Support

## Implemented

- `_Generic` - Type-generic selection
- `_Alignas` / `_Alignof` - Alignment specifiers
- `_Atomic` - Atomic types and operations
- `_Thread_local` - Thread-local storage
- `_Noreturn` - Function specifier for non-returning functions
- `_Static_assert` - Compile-time assertions
- `_Bool` - Boolean type
- Anonymous structs/unions
- `u8` string literals (UTF-8)
- `__STDC_NO_*` feature test macros
- `char16_t` and `char32_t` types (built-in typedefs)
- `<threads.h>` support (via system headers)
- Unicode string literals (`u""`, `U""`, `u8""`, `L""`)
- Complex Number types

## Out of Scope

### Bounds-checking interfaces (Annex K)
Optional `*_s` functions like `strcpy_s`, `sprintf_s`.
This is a C library feature, not a compiler feature. glibc intentionally does not implement Annex K.
